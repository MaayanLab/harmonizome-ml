# README

## Setup
Install javascript
```
npm install
```

## Development
### Setup
**Setup JS dependencies**
```
npm install
```
**Setup Python dependencies**
```
pip install -r requirements.txt
```

### Debugging
**Start JS script generator watchdog and python flask server**
```
grunt start
```

## Production
**Build templates**
```
grunt build
```
**Start Production server**
```
docker-compose up
```

## Gradle
```
gradle --no-daemon runProject
```
